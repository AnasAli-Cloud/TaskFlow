# TaskFlow - Role-Based Task Management System

## Overview

TaskFlow is a web-based task management system developed using PHP, MySQL, HTML, and CSS. The application allows administrators to create, assign, edit, and manage tasks, while users can view and update the status of their assigned tasks.

## Features

* User Authentication and Session Management
* Role-Based Access Control (Admin & User)
* Create, Read, Update, and Delete (CRUD) Tasks
* Task Assignment to Users
* Task Status Tracking (Complete / Incomplete)
* Task Filtering (All, Pending, Completed)
* Progress Tracking Dashboard
* Due Date Management
* Overdue Task Detection

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* WampServer
* SQL

## Project Structure

* index.php – Login Page
* dashboard.php – Main Dashboard
* add_task.php – Add New Tasks
* edit_task.php – Edit Existing Tasks
* logout.php – Logout Functionality
* db.php – Database Connection

## Installation

1. Clone the repository.
2. Place the project folder inside the WampServer www directory.
3. Import the database SQL file into MySQL.
4. Update database credentials in db.php.
5. Start Apache and MySQL.
6. Open the project in your browser.
